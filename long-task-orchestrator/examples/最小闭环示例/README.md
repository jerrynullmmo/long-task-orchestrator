# 最小闭环示例

本目录演示一个完全脱敏的长任务闭环：

1. 主线程写入 `状态包/当前状态.json` 和 `状态包/调度记录.json`。
2. 主线程创建 `状态包/任务包/文档整理任务.json`。
3. 独立执行单元写入 `状态包/回执/文档整理任务回执.json`。
4. 监控器写入 `状态包/监控/文档整理任务监控.json`。
5. 主线程运行校验脚本后决定是否放行。

运行：

```bash
python3 ../../scripts/validate_orchestrator.py --project .
```
